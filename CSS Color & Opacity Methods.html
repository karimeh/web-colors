🎨 CSS Color & Opacity Methods

From simplest named colors to advanced relative color syntax — with live examples.

🏷️

1. Named Colors

CSS supports 148 named colors. These are the easiest way to apply colors, though the palette is limited.

Red
color: red;
RebeccaPurple
background: rebeccapurple;
CornflowerBlue
border-color: cornflowerblue;
💡 Named colors include transparent and currentColor (covered later).
🔢

2. HEX Colors

Hexadecimal (HEX) notation is the most common. Supports shorthand (#RGB) and full (#RRGGBB). No alpha in shorthand, but modern browsers support 8-digit hex with alpha (#RRGGBBAA).

#FF5733
background: #FF5733;
#0F0 (shorthand)
color: #0F0; /* = #00FF00 */
#33415580 (with alpha)
background: #33415580; /* 50% opacity */
📊

3. RGB / RGBA

RGB uses red, green, blue values (0–255). RGBA adds an alpha channel (0–1). Modern syntax also allows space-separated values and alpha with slash.

SteelBlue
background: rgb(70, 130, 180);
Tomato 70% opacity
background: rgba(255, 99, 71, 0.7);
Green (modern syntax)
color: rgb(0 255 0 / 0.3);
🌈

4. HSL / HSLA

HSL defines colors by hue (0–360), saturation (0–100%), and lightness (0–100%). HSLA adds alpha. Intuitive for creating color variations.

Vibrant Blue
background: hsl(200, 100%, 50%);
Pink 60% opacity
background: hsla(340, 80%, 60%, 0.6);
Green (modern syntax)
border: 3px solid hsl(120 100% 40% / 0.5);
🎚️

5. HWB

HWB (Hue, Whiteness, Blackness) is another cylindrical model. Easier to lighten/darken by adjusting whiteness/blackness.

Red (pure)
background: hwb(0 0% 0%);
Muted Blue
color: hwb(200 30% 20%);
Purple 70% opacity
background: hwb(300 10% 20% / 0.7);
🔬

6. CIE Lab & LCH

Lab and LCH provide perceptually uniform color spaces. Lab uses Lightness, a-axis (green↔red), b-axis (blue↔yellow). LCH uses Lightness, Chroma, Hue.

Mid Gray (Lab)
background: lab(50% 0 0);
Violet-ish
color: lab(60% 60 -40);
Orange (LCH)
background: lch(70% 70 40);
⚠️ Lab/LCH values can go beyond sRGB gamut; they may be clipped on standard displays.
🧪

7. Oklab & Oklch

Oklab and Oklch are improved perceptual color spaces, designed for better uniformity and fewer hue shifts when interpolating.

Warm Pink
background: oklab(0.6 0.15 0.1);
Sky Blue
color: oklch(0.7 0.2 250);
Green 80% opacity
background: oklch(0.5 0.25 140 / 0.8);
🖥️

8. color() – Wide‑Gamut

The color() function allows specifying colors in various color spaces (e.g., display-p3, srgb, a98-rgb). Ideal for modern HDR screens.

sRGB Red
background: color(srgb 1 0 0);
Display‑P3 Orange
color: color(display-p3 1 0.5 0);
sRGB Blue 60% opacity
background: color(srgb 0.2 0.6 0.8 / 0.6);
👻

9. Opacity & Filter: Opacity()

The opacity property makes an entire element (including children) transparent. filter: opacity() can apply opacity to just the element itself (but still affects layering).

RGBA alpha = 0.5
background: rgba(255,0,0,0.5);
Text
element opacity:0.5
opacity: 0.5; background: red;
filter: opacity(50%)
filter: opacity(50%); background: #3b82f6;
♻️

10. currentColor & transparent

currentColor refers to the element’s text color. transparent is fully transparent (equivalent to rgba(0,0,0,0)).

currentColor (inherits text color)
background: currentColor;
transparent
background: transparent;
currentColor + opacity
background: currentColor; opacity: 0.4;
🖥️

11. System Colors

System colors reflect the user’s operating system theme (light/dark). Useful for respecting user preferences.

Canvas
background: Canvas; color: CanvasText;
Highlight
background: Highlight;
GrayText
color: GrayText;
System colors react to OS theme changes; they are powerful for accessible design.
🎛️

12. CSS Variables & Opacity

Define colors once with custom properties; reuse with different opacities by decomposing channels or using color-mix().

Original Brand Color
background: var(--brand);
Brand at 40% opacity (relative syntax)
background: rgb(from var(--brand) r g b / 0.4);
Brand mixed with transparent
background: color-mix(in srgb, var(--brand) 60%, transparent);
🧴

13. color-mix()

color-mix() blends two colors in a specified color space. Perfect for creating tints, shades, or opacity effects without losing gamut.

Red + Blue (50% each)
background: color-mix(in srgb, red 50%, blue);
Emerald tint (lch space)
background: color-mix(in lch, #10b981 70%, white);
Navy 80% opacity (via transparent)
background: color-mix(in oklch, navy 80%, transparent);
🧬

14. Relative Color Syntax

Transform a color by modifying individual channels using from. The ultimate tool for dynamic theming and color manipulation.

Violet at 50% alpha
background: rgb(from #8b5cf6 r g b / 0.5);
Green lightened by 20%
background: hsl(from #059669 h s calc(l + 20));
Magenta desaturated (oklch)
background: oklch(from #d946ef l calc(c * 0.5) h);

✨ All examples render live using your browser’s CSS engine. For latest support, check caniuse.com.
